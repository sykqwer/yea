# Портфолио Владимир Шелекасов

Многоязычный сайт-визитка с переключением тем и формой обратной связи.

## Функционал

- 🌓 Переключение светлой/тёмной темы
- 🌐 Русский/Английский языки (определяется автоматически по браузеру)
- 📄 Экспорт резюме в PDF
- 📱 Полностью адаптивный дизайн
- 📝 Форма обратной связи с валидацией
- ⚡ SPA-подобная навигация с 404 страницей

## Технологии

- HTML5 / CSS3 (CSS Grid, Flexbox)
- JavaScript (ES6+)
- Font Awesome Icons
- LocalStorage для сохранения темы

git clone https://github.com/sykqwer/yea.git
